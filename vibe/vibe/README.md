# AMORA.AI Project

## File Setup Instructions

### For the About Page Animation:

1. **Add baby-girl.gif:**
   - Place your `baby-girl.gif` file in the `/public/` folder
   - The file path should be: `/public/baby-girl.gif`

2. **Add Audio File:**
   - Create an audio file saying "Hello! I want to know you more"
   - Save it as either:
     - `/public/hello-audio.mp3` (MP3 format)
     - OR `/public/hello-audio.wav` (WAV format)
   - You can use text-to-speech tools or record your own audio

### For the Landing Page:

1. **Add Floating Images:**
   - Place your 6 images in `/public/images/` folder
   - Name them: `image1.jpg`, `image2.jpg`, `image3.jpg`, `image4.jpg`, `image5.jpg`, `image6.jpg`
   - Or update the image paths in `index.html` to match your file names

## Features

- Landing page with "AMORA.AI" in uppercase
- Lavender and pearl gradient background
- Floating images with animations
- About page with animated baby-girl.gif
- Audio greeting on the about page
- Responsive design

## Running the Project

Use any local web server to view the project. The HTML files include auto-reload scripts for development.
